Final_Project — COVID-19 Data Scraper and Analysis
Overview:
This project focuses on the collection, storage, and analysis of COVID-19 statistics from a government-run website. I developed a Python-based scraper to extract up-to-date information and used SQLite3 to organize and query the data efficiently. The analysis includes computing average cases, total cases, and standard deviation of reported cases across North America compared to global figures. The results were visualized using Matplotlib to identify regional trends and differences.
Implications:
This project demonstrates my ability to handle real-world datasets, implement data pipelines, and derive meaningful insights through statistical analysis and visualization. It showcases my practical understanding of data processing, database management, and analytical reasoning.

Doctor_Tree — Hierarchical Tree Data Structure
Overview:
In this project, I implemented a hierarchical tree data structure from scratch to represent ordered relationships between data elements. The implementation supports standard traversal operations, including preorder, inorder, and postorder traversals, each of which provides a different method of accessing and displaying the tree’s contents. The structure also includes an insertion function to dynamically build and expand the tree.
Implications:
Doctor_Tree demonstrates my understanding of fundamental data structures and recursive algorithms. It emphasizes my ability to design efficient, well-organized code and to implement key computational concepts without relying on external libraries.

Emergency_Queue — Custom Heap Implementation
Overview:
This project involves creating a priority queue system modeled as a custom min-heap. The implementation includes the core heap operations: heapify up, heapify down, insert, peek, print, and remove_min. The data structure maintains the heap property at all times, ensuring that the element with the highest priority (lowest value) can be accessed in constant time.
Implications:
Emergency_Queue highlights my ability to implement efficient data structures and algorithms from first principles. The project demonstrates a strong understanding of time complexity, priority queue logic, and the importance of selecting appropriate data structures for specific computational tasks.
